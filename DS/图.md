1. 一个==单独的顶点==**也算一个连通分量**
2. 连通分量是==极大连通子图==，生成树是==极小连通子图==
3. 区别连通图和环，连通图可能有环
4. 无向图的邻接表中每条边在表中存两次
5. 对于图的题目，看清是无向图还是有向图，深度优先还是广度优先
6. 非强连通图可能只用一次 BFS 遍历。非连通图至少 2 次 BFS 遍历，非强连通图至少 1 次 BFS 即可遍历
7. 强连通图：有向图中每个顶点之间均有路径，注意是路径，不一定有边（区别 u 到 v 的路径和 u 到 v 的路径边）
8. 关键路径的最早时间与最晚时间
	1. 事件（顶点Vk）最早开始时间
		- 从前到后计算
		- 源点为 0
		- Max { 前一个事件（顶点Vj）+ Weight(Vj, Vk) }
	2. 事件（顶点Vk）最晚开始时间
		- 从后到前计算
		- 汇点 = 其最早开始时间
		- Min { 前一个事件（顶点Vj）+ Weight(Vj, Vk) }
	3. 活动（弧<Vj, Vk>）最早发生时间
		- 只关注弧尾（前一个事件）Vj
		- 最早开始时间 = Vj 的最早开始时间
	4. 活动（弧<Vj, Vk>）最晚发生时间
		- 关注弧头（后一个事件）Vk 与 Weight(Vj, Vk)
		- 最晚发生时间 = Vk 的最晚发生时间 - Weight(Vj, Vk)
9. 存在相同权值边的图的MST可能也唯一
	- 带权连通图MST唯一的充分条件：不存在相同权值的边
	- 充要条件：任意一个环包含的边的权值都不等
10. 拓扑序列唯一性结论
	- 有向图邻接矩阵为三角矩阵，其拓扑序列存在，但可能不唯一
	- 有向无环图的拓扑序列唯一，其对应的图不一定唯一
