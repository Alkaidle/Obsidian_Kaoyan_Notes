```c
		// 将p后的部分逆置(不包括p)
		q = p->next;
		p->next = NULL;
		while(q->next!=NULL) {
			r = q->next; // 先保存要断开的结点后面的
			q->next = p->next; // 插入到头后面，交错进行
			p->next = q;
			q = r;
		}
		
		//将p所指的整个链表逆置
		q = p->next;
		p->next = NULL;
		while(q->next!=NULL) {
			r = q->next;
			q->next = p;
			p = q;
			q = r;
		}
		q->next = p;
```

- ==循环队列==
    
    - **rear 指向队尾元素下一个位置**
        
        - 初始：**front=rear=0**
            
        - 判空：front=rear
            
        - 判满：
            
            - 牺牲一个存储单元：(rear+1)%n=front
                
            - 增加size：front=rear 与 size
                
            - 增加tag，删除 tag=0，插入 tag=1：front=rear 与 tag
                
    - **rear 指向队尾元素**
        
        - 初始：**front=0**，**rear=n-1**
            
        - 判空：**(rear+1)%n=front**
            
        - 判满：
            
            - 牺牲一个存储单元：(rear+**2**)%n=front
                
            - 增加size：front=rear + size
                
            - 增加tag，删除 tag=0，插入 tag=1：front=rear + tag

- ==表达式树==
    
    - ==根据表达式建立== **左右操作数分别在左右节点**，优先级最高先建子树
        
    - ==根据后缀表达式==
        
        - **从左到右**扫描后缀式
            
        - 遇操作数 → 建节点并压栈
            
        - 遇运算符 → 弹出两个节点（**右操作数先弹**），建立以该运算符为根的新子树，再压栈
            
        - 扫描结束后，栈顶节点即为整个表达式树根
            
    - ==根据前缀表达式==
        
        - **从右到左**，**左操作数先弹**
            
- ==KMP 串的模式匹配==
    
    - **核心**：在模式串 P 第j个位置不匹配跳转到 P 的 next[j] 位置处继续匹配，若不是 P 的第1个不匹配，则主串指针不动
        
    - 计算 next[j]：next[1]=0, next[2]=1, 其余为**部分匹配值** PM[j-1]+1  
        部分匹配值：前缀和后缀最长公共位数
        
    - ①若用 next 数组确定跳转位置，根据当前模式串中**第 j 个不匹配的 next[j]**，②若直接根据前缀确定右移位数则 j-1-PM[j-1]
        
    - **优化**：if (P[j]=P[next[j]])，递归 nextval[j]=nextval[next[j]]
        
    - 先算 PM，其整体加1（序号从1开始，从0开始就不加）
        
    - **主串不回溯，即序号不减**
        
    - ==KMP 右移位数==
        
        - 可直接看
            
        - j - next[j]
            
        - j-1 - PM[j-1]
