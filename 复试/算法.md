#### 滑动窗口
##### 01 定长
窗口右端点在 i 时，窗口长度为 k，则窗口左端点为 i−k+1
1. **入**：下标为 i 的元素进入窗口，更新相关统计量。如果窗口左端点 i−k+1<0，则尚未形成第一个窗口，重复第一步。
2. **更新**：更新答案。一般是更新最大值/最小值。
3. **出**：下标为 i−k+1 的元素离开窗口，更新相关统计量，为下一个循环做准备。

> [!NOTE] 
> 不用指明左端点，可用`left=i-k+1`，左端点离开窗口时直接对下标为left的进行修改

#### 反转链表
##### 01 反转整个链表
[206. 反转链表](https://leetcode.cn/problems/reverse-linked-list/description/?envType=study-plan-v2&envId=top-100-liked)
```c++
ListNode* reverseList(ListNode* head) {
        ListNode *pre=nullptr, *cur=head, *nxt;
        while (cur) {
            nxt=cur->next;
            cur->next=pre;
            pre=cur;
            cur=nxt;
        }
        // 反转后pre为头节点，cur为空指针
        return pre;
    }
```
##### 02 反转区间
[92. 反转链表](https://leetcode.cn/problems/reverse-linked-list-ii/description/)
在头节点之前新增一个prehead结点，防止left为1
注意反转结束后pre和cur的最终状态
![[Pasted image 20260112214533.png]]
```c++
ListNode* reverseBetween(ListNode* head, int left, int right) {
        ListNode prehead(0, head), *p0=&prehead;
        // 注意left为1的情况
        for (int i=0; i<left-1; i++) {
            p0=p0->next;
        }
        ListNode *pre=nullptr, *cur=p0->next, *nxt;
        for (int i=0; i<right-left+1; i++) {
            nxt=cur->next;
            cur->next=pre;
            pre=cur;
            cur=nxt;
        }
        // 注意反转结束后pre和cur的最终状态
        p0->next->next=cur;
        p0->next=pre;
        // prehead没有变，而head指向的结点可能不再是头节点
        return prehead.next;
    }
```
##### 03 k个一组反转
[25. K 个一组翻转链表](https://leetcode.cn/problems/reverse-nodes-in-k-group/description/?envType=study-plan-v2&envId=top-100-liked)
```c++
ListNode* reverseKGroup(ListNode* head, int k) {
        ListNode *cur=head;
        // 链表长度
        int n=0;
        while (cur) {
            n++;
            cur=cur->next;
        }
        ListNode prehead(0, head);
        ListNode *p0=&prehead;
        ListNode *pre=nullptr;
        cur=p0->next;
        while (n>=k) {
            n-=k;
            // 反转区间
            for (int i=0; i<k; i++) {
                ListNode *nxt=cur->next;
                cur->next=pre;
                pre=cur;
                cur=nxt;
            }
            // p0->next 为反转前的区间头，反转后的区间尾，反转后pre为区间头
            // 反转后区间尾(p0->next)为下一个区间的p0，需要保存
            ListNode *nxt=p0->next;
            p0->next->next=cur;
            p0->next=pre;
            p0=nxt;
        }
        return prehead.next;
    }
```

ListNode *detectCycle(ListNode *head) {
        ListNode *p=head, *q=head;
        while (p && p->next) {
            p=p->next->next;
            q=q->next;
            if (q==p) {
	            // 头指针到相遇点=慢指针从相遇点到入口的距离+若干倍环长
	            // 头指针和慢指针相遇
                while (q!=head) {
                    head=head->next;
                    q=q->next;
                }
                return q;
            }
        }
        return NULL;
    }

#### 前后指针
[19. 删除链表的倒数第 N 个结点](https://leetcode.cn/problems/remove-nth-node-from-end-of-list/description/?envType=study-plan-v2&envId=top-100-liked)
```c++
// q指向倒数第n个结点
ListNode* removeNthFromEnd(ListNode* head, int n) {
        // 可能删除头节点时需要设置prehead
        ListNode prehead(0, head);
        ListNode *p=&prehead, *q=&prehead;
        for (int i=0; i<n; i++) {
            p=p->next;
        }
        while (p->next) {
            p=p->next;
            q=q->next;
        }
        q->next=q->next->next;
        return prehead.next;
    }
```
#### 相交链表
[160. 相交链表](https://leetcode.cn/problems/intersection-of-two-linked-lists/description/?envType=study-plan-v2&envId=top-100-liked)
```c++
ListNode *getIntersectionNode(ListNode *headA, ListNode *headB) {
        ListNode *p=headA;
        ListNode *q=headB;
        // y+(x+z)=x+(y+z)
        while (p!=q) {
            p=p ? p->next : headB; // 判断p是否为空
            q=q ? q->next : headA;
        }
        return p;
    }
```

        while (p && p->next) {

            q=q->next;

            ListNode* nxt=cur->next;

bool isPalindrome(ListNode* head) {
        ListNode *mid=middleNode(head);
        ListNode *h2=reverseList(mid);
        // h2与h2前面的链不会断，长度为偶数时前半部分多一个
        while (h2) {
            if (head->val!=h2->val) {
                return false;
            }
            head=head->next;
            h2=h2->next;
        }
        return true;
    }

#### 回溯
##### 01 子集型回溯
[78. 子集](https://leetcode.cn/problems/subsets/description/?envType=study-plan-v2&envId=top-100-liked)
- 当前操作（如选/不选）
- 子问题
- 下一个子问题
输入角度：
- 当前元素选/不选
- 下标≥i的数字中选
- 下标≥i+1的数字中选
答案角度：
- 下标j≥i，加入路径
- 子问题：从下标≥i数字中构造
- 下一个子问题：从下标≥j+1数字中构造
```c++
vector<vector<int>> ans;
vector<int> t;

vector<vector<int>> subsets(vector<int>& nums) {
        int n=nums.size();
        for (int m=0; m<(1<<n); m++) {
            t.clear();
            for (int i=0; i<n; i++) {
                if (m&(1<<i)) {
                    t.push_back(nums[i]);
                }
            }
            ans.push_back(t);
        }
        return ans;
    }    
```
