### 数组求最大值
- 基本
`auto it = max_element(v.begin(), v.end());`
`int mx = *it;`
`int idx = it - v.begin();`
- 自定义
```c++
int mx = *max_element(v.begin(), v.end(),
                      [](int a, int b) {
                          return abs(a) < abs(b);
                      });
```
- 二维
```c++
vector<vector<int>> a = {{1,3}, {5,2}, {4,9}};

int mx = INT_MIN;
for (auto &row : a)
    mx = max(mx, *max_element(row.begin(), row.end()));
```
