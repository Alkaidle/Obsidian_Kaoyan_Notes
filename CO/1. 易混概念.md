| 周期名称                     | 含义                   | 主要作用对象       | 层次关系                            |
| ------------------------ | :------------------- | ------------ | ------------------------------- |
| **时钟周期 **                | 最小时间单位               | CPU 内部       | 主频的倒数，又称节拍，最小计时单位，CPU 内部定时信号    |
| **机器周期 / CPU 周期 / 工作周期** | CPU 完成一个基本操作         | CPU          | 由若干时钟周期组成                       |
| **指令周期**                 | 完成一条指令的全部过程          | CPU          | 由多个机器周期组成，最大单位<br>每个阶段包含多个机器周期  |
| **总线周期**                 | 一次总线读写操作             | CPU ↔ 外设/存储器 | 若干时钟周期<br>CPU ↔ 主存时，**等同于存取周期** |
| **存取周期**                 | 主存完成一次完整访问（读/写 + 恢复） | 主存           | 决定存储器速度                         |
