1. 创建进程：用户登录，作业调度（高级调度），系统提供服务，用户程序请求
2. 单处理器中10个进程，**就绪队列**最多9个，**阻塞队列**最多10个
3. 程序的封闭性指程序执行结果只取决于程序本身，不受外界影响
4. ==CPU 状态==是指 CPU 内部寄存器的信息，也称 CPU 上下文，需要**记录在 PCB 中**
   内核态和用户态是指 ==CPU 运行模式==，用 PSW 记录
5. 线程包含 CPU 现场，可以独立执行程序（因为是调度的基本单位）
6. 线程虽然有自己的存储区，但地址空间不是独立的，是属于进程地址空间的
7. 对于键盘这种慢速 I/O 分配一个线程即可
8. 全局变量是相对同一进程而言的，两个进程无法通过全局变量交换数据；可以通过文件系统：管道
