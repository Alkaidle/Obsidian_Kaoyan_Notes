1. 硬件变换地址机构用于动态重定位，静态重定位用 OS 或装入程序地址变换
2. 页表在进程创建时就分配在主存，进程被调度运行时，操作系统把该进程的 **页表基址寄存器（PTBR）** 设置为该进程页表在主存的地址。可以说页表常驻内存能加速虚实地址转换
3. 虚拟存储器==最大容量==由 **CPU 地址位数决定**，和主存、外存无关，==实际可用容量==受 **主存大小 + 外存（交换空间）大小** 限制，如果程序使用的虚拟内存超出主存+外存之和，就会内存不足，导致进程被终止
4. 驻留集可定义为“分配给进程的页框集合“（分配视角）也可以定义为“进程当前实际驻留的页面集合”（实际驻留视角）

5. 固定分区分配中每个分区大小可以不同但预先固定（确定了不能改变）
6. 地址变换机构（通常由 **基址寄存器** + **界限寄存器** 组成）放在 CPU 的内存管理单元（**MMU**）里
7. 页式系统中*形成有效地址、查页表、拼接物理地址* → 由**硬件**（**MMU**）自动完成
8. ==可重入程序==是通过共享使用同一块存储空间，或通过动态链接将所需的程序段映射到相关进程中，优点是减少了程序段的调入调出，减少了对换数量
9. ⚠区分
	- 对主存的访问：字或字节为单位
	- 对主存的分配：页、段
