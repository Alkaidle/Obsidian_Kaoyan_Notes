1. *异构网络*：不同物理介质/数据链路层协议/网络层协议
2. **虚电路**
	- 有连接、可靠或不可靠（具体看协议设计）
	- 连接可分为：
		- 永久性虚电路 PVC：提前定义好，基本不需要任何建立时间
		- 交换型虚电路 SVC：临时建立连接
	- 逻辑连接，不需要预分配带宽（物理上）
3. IP 四个重要协议：IP、ICMP、ARP、RARP（逆地址解析）
4. ⚠区分：IP 分组中的协议和版本字段
	- 协议：使用 IP 的上层协议
	- 版本：IP 版本
5. ⚠区分
	- IP 首部校验和**不能检验某个具体字段是否出错**，只能校验整个首部是否有错
	- IP **不要求重传**有差错的分组，但会通过 ICMP 报告异常：**校验和发现错误 → IP 协议栈进一步定位问题 → ICMP 报告异常**（ICMP 差错报文中参数问题类型，首部中某些而非任意字段有问题才发送 ICMP）
6. “路由器不会正常转发的地址”
	- 本网络内部使用的地址
	- 环回地址
	- 私有地址
	- 组播地址（普通路由器不会作为单播转发）

8. **子网划分**
	- “将一个 IP 网络（没给）划分为4个子网”：找节点中前缀最小的子网（如/25），以其前一位作为根节点（以/24作为根节点），要求所有划分的节点能覆盖整个哈夫曼树（左0右1）
	- “将某**给定的** IP 地址划分为5个子网”：以该 IP 地址为根节点，要求划分的子网覆盖**以该节点为根**的哈夫曼树
	- 要使划分的子网可分配主机数最少：树的高度尽可能高，除最后一个节点外每个jie'dian
