1. *异构网络*：不同物理介质/数据链路层协议/网络层协议
2. **虚电路**
	- 有连接、可靠或不可靠（具体看协议设计）
	- 连接可分为：
		- 永久性虚电路 PVC：提前定义好，基本不需要任何建立时间
		- 交换型虚电路 SVC：临时建立连接
	- 逻辑连接，不需要预分配带宽（物理上）
3. IP 四个重要协议：IP、ICMP、ARP、RARP（逆地址解析）
4. ⚠区分：IP 分组中的协议和版本字段
	- 协议：使用 IP 的上层协议
	- 版本：IP 版本
5. ⚠区分
	- IP 首部校验和**不能检验某个具体字段是否出错**，只能校验整个首部是否有错
	- IP **不要求重传**有差错的分组，但会通过 ICMP 报告异常：**校验和发现错误 → IP 协议栈进一步定位问题 → ICMP 报告异常**（ICMP 差错报文中参数问题类型，首部中）
