1. 实时性要求高→无连接
2. 数据链路层**提供可靠传输**（和流量控制交织在一起）
3. 循环校验码计算冗余码时后面要添r个0（生成多项式r+1位），带r个检验位的多项式编码可以检测到所有长度小于等于r的突发性错误
4. GBN 协议中，已经发送但未确认、也未超时的帧不能再次发送
5. 算信道利用率，题目中说了确认帧长度则要考虑确认帧传输时延

#### 介质访问控制
---
1. 集中式介质访问控制：主站控制从站
2. **TDM 原理和特点**
	- TDM 在发送端将不同用户的信号交织在不同时间片内，接收端提取各时间片内的信号
	- 实现 TDM 的条件：
		- 介质位速率大于单个信号的位速率
		- 介质带宽大于结合信号的带宽（所有信号经调制后形成的复合信号）
	- TDM 与 FDM 相比，抗干扰能力强，可以逐级再生整形，避免干扰积累，TDM 适合传输数字信号，FDM 适合传输模拟信号
3. 码分复用：**不同站之间**码片规格化内积为 0，接收方规格化内积得到 **-1**，表示**发送的数据为 0**
4. “两个站点之间的距离” → **单程**，“使用 Hub 连接站点”：**两段链路**
5. 没有数据要发送的节点收到令牌后**立即转发**
6. 以太网中最短帧长固定为 64B，而非争用期（根据不同速率改变）
7. 信道预约 RTS 帧和 CTS 帧指定的持续时间是“从收到...后开始算的”，发送数据帧前等待 **SIFS**，（发送 RTS 前等待 DIFS）
